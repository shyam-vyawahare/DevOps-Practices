## Common Commands

- ps aux  
  Lists all running processes.

- top  
  Displays real-time system resource usage.

- htop  
  Interactive process viewer (if installed).

- kill <PID>  
  Terminates a process by Process ID.

- uptime  
  Shows system running time and load average.

Understanding process monitoring is essential
for diagnosing performance issues in production systems.
